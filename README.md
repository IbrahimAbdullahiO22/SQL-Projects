# SQL Projects Portfolio
A collection of hands-on SQL analysis projects built as part of my self-directed learning journey in data analysis and database querying.

---

## About This Portfolio

This repository demonstrates practical SQL skills applied across six different datasets spanning retail, music, business, healthcare, and event management. Each project explores real analytical questions using structured query techniques — from basic filtering to multi-table JOINs, subqueries, and conditional logic.

---

## Projects

### 1.  Chinook Digital Media Store Analysis
Analysis of a digital media store database including artists, albums, tracks, invoices, and customers.

**Skills:** Multi-table JOINs (up to 4 tables), aggregations, GROUP BY, ORDER BY, date filtering, DISTINCT, LIKE

**Key Questions Answered:**
- Which sales agent generated the most revenue in 2009?
- Which country purchases the most items?
- What is the most popular genre by tracks sold?
- Which customer has the highest lifetime spend?

---

### 2.  Climate Conference VIP Analysis
SQL analysis of VIP participation, hotel reservations, event attendance, and feedback logistics at a global climate summit.

**Skills:** INNER JOIN, LEFT JOIN, Self JOIN, Subqueries, GROUP BY, NULL filtering, multi-condition WHERE filtering

**Key Questions Answered:**
- Which VIPs have no hotel reservation?
- How many VIPs RSVP'd yes to dinner per event?
- Which government VIPs are attending events?
- Who has been a member longer than the average membership year?
- Which hotel has the most reservations?

---

### 3.  Customer & Order Analysis
Explores customer orders, product sales, and revenue patterns for a fictional retail system using SQLite.

**Skills:** JOINs, aggregate functions, GROUP BY, HAVING, WHERE filtering, data cleaning with length()

**Key Questions Answered:**
- How many orders were placed in January vs February?
- Which product brought in the most revenue in January?
- What was the average amount spent per account in February?
- Which New York locations received at least 3 orders?

---

### 4.  Fortune 500 Workplace Benefits & Financials
SQL analysis of Fortune 500 company data covering revenue, employee count, healthcare benefits, PTO, and maternity leave policies.

**Skills:** CASE statements, HAVING, GROUP BY, conditional filtering, aggregate functions

**Key Questions Answered:**
- Which industry offers the best average maternity leave?
- Which industries have the highest average employee tenure?
- Which companies offer healthcare benefits AND more than 20 days PTO?
- How are companies rated based on maternity leave quality?

---

### 5.  Spotify Top 50 Songs 2021 Analysis
Analysis of Spotify Top 50 tracks using audio features including danceability, energy, tempo, and popularity.

**Skills:** Table creation and schema design, aggregate functions, GROUP BY, WHERE filtering, audio feature analysis

**Key Questions Answered:**
- Which artist has the most songs in the Top 50?
- Which track has the highest energy level?
- What are the top 5 most popular tracks?
- Which artist has the highest average popularity score?

---

### 6.  Superstore Sales & Inventory Analysis
Analysis of sales and inventory data from a mock Superstore database across Electronics, Furnishings, Appliances, and Kitchen Supplies.

**Skills:** Aggregate functions (AVG, SUM, MIN, MAX, COUNT), GROUP BY, ORDER BY, WHERE filtering, inventory analysis

**Key Questions Answered:**
- Which category has the highest total inventory value?
- Which item has the highest customer rating?
- What is the average price per category?
- Which items are expensive and low in stock?
